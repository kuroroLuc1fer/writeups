# Penetration Testing & CTF Writeups

Welcome to this repository! 🔒 Here you'll find detailed writeups of my penetration tests on various labs and my participation in Capture The Flag (CTF) competitions.

## 📚 About

This repository centralizes my learning documentation in cybersecurity and penetration testing. Each writeup details the exploitation process, identified vulnerabilities, and techniques used.

## 📂 Repository Structure

Each writeup is organized and typically contains:
- **Description**: Objective of the lab/CTF
- **Approach**: Reconnaissance and exploitation methodology
- **Vulnerabilities Found**: CVEs, misconfigurations, etc.
- **Step-by-Step Solution**: Relevant commands and scripts
- **Lessons Learned**: Key points and security recommendations

## 🎯 Writeup Categories

- **HackTheBox**: Machines from the HTB platform
- **TryHackMe**: Boxes and rooms from TryHackMe
- **CTF Events**: Competitions and CTF events
- **Custom Labs**: Personal testing environments

## 🔧 Tools & Techniques

The writeups cover the use of tools such as:
- Reconnaissance: `nmap`, `masscan`, `gobuster`
- Exploitation: `Metasploit`, `SQLmap`, custom scripts
- Post-exploitation: `privilege escalation`, `lateral movement`
- Web: `Burp Suite`, `OWASP Top 10`

## 📖 How to Use This Repository

1. **Navigate**: Browse folders by platform or difficulty
2. **Read**: Check the `.md` files for details on each writeup
3. **Learn**: Adapt techniques to your own labs and environments
